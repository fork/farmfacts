# dullahan

dullahan is the FarmFacts file manager component. It uses WebDAV for
communication.

## Setup

    $ git clone git@github.com:fork/farmfacts.git
    $ cd farmfacts/dullahan
    $ bundle install

Then start a Rack compatible server of your choice, mine was:

    $ passenger start
